# All the News That's Fit to Scrape
In this app, it will "scrape" a news article headlines.  A user can "scrape" to get the latest artcle headlines, read the article, and make a comment.

## Built With

### Technologies Used
- Express
- Express Handlebars
- MVC (Model View Controller)
- JQuery
- Bootstrap
- Mongo/Mongoose
- Node
- Cheerio
- Axios
